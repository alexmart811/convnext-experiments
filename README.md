# convnext-experiments

Репозиторий для экспериментов с архитектурой **ConvNeXt** из статьи [*A ConvNet for the 2020s*](https://arxiv.org/abs/2201.03545). Здесь собираются различные сценарии обучения и сравнения моделей на разных задачах и датасетах

## Эксперименты

Ниже перечислены каталоги с экспериментами (подробнее об эксперименте смотреть в README файлах в соответствующей директории):

- **`convnext_experiments/datasets_comparison_benchmark/`** — **Автор:** Мартынов Александр. Сравнение работы ConvNeXt на датасетах классификации и сегментации; детали запуска и конфигурации смотрите в коде и readme внутри каталога при необходимости.
- ...

## Работа с проектом (uv)

Чтобы подтянуть зависимости:

```bash
uv sync
```

Для зависимостей разработки:

```bash
uv sync --group dev
```

Команды выполняют из корня репозитория. Дальнейший запуск скриптов — через `uv run …` или активировав `.venv` после синхронизации.
